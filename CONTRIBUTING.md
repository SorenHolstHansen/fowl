# Contribution guide

It would be absolutely amazing if you would want to contribute to the development of Fowl! Anything big or small goes a long way, and I am really just pleased you are here.

## Requirements

Before you can get started contributing to fowl, you will need the following.

- [Rust](https://rust-lang.org/)
- [re2c version 4](https://re2c.org/)

## Getting started

A simple way to get started is to cd into one of the examples and try to run it, e.g.

```
cd examples/simple && cargo run -- run
```
